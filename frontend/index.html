02

The language · ~7 min

Python in 7 minutes

Three ideas: variables, lists, and DataFrames. Everything else follows from these.

⏱ ~7 minutes · follow along, then experiment

Python reads almost like English, designed to be understandable by humans first, computers second.

Cell 2 — Variables & lists
basics.py
# A variable is a labelled box that stores a value university = "UIUC" semester = "Fall 2024" total_courses = 6 print(university) # → UIUC print(total_courses * 3) # → 18 (credit hours) print(f"Semester: {semester}") # → Semester: Fall 2024 # A list holds multiple values in order courses = ["ART 101", "CS 101", "MATH 231", "GD 210", "ID 310"] grades = [3.7, 3.3, 2.7, 4.0, 3.0] print(courses[0]) # → "ART 101" (counting starts at 0!) print(len(grades)) # → 5 print(sum(grades) / len(grades)) # → 3.34 (semester GPA)
Cell 3 — The DataFrame
dataframe.py
# A DataFrame is a spreadsheet, simply columns of related data # Each row here is one UIUC student, each column is a fact about them data = { "name": ["Aditya", "Jordan", "Sam", "Taylor", "River"], "major": ["Ind. Design", "Graph. Design", "Arch.", "Graph. Design", "Ind. Design"], "year": ["Junior", "Sophomore", "Senior", "Junior", "Freshman"], "gpa": [3.7, 3.2, 3.9, 3.5, 3.4], "credits": [85, 45, 112, 78, 15] } df = pd.DataFrame(data) print(df) print(df["gpa"].mean()) # → avg GPA across all students print(df.describe())
💡
Mental model: Think of a DataFrame like a spreadsheet in Illustrator's data variables panel. Each column is a category of info, each row is one student. Almost everything in data analysis is transforming this table.
03

Visualization · ~8 min

Your first chart

Two lines of code. A real chart. Then we'll make it beautiful.

⏱ ~8 minutes · run each cell, discuss what changes
Cell 4 — The simplest possible chart
first_chart.py
sns.set_theme(style="whitegrid", palette="muted") sns.barplot(data=df, x="name", y="gpa") plt.show()

Now add color to encode a second variable. Each visual property (position, size, color, shape) can carry a dimension of data.

Position (x) → student name
Height (y) → GPA
Color → major
Cell 5 — Color as information
color_chart.py
fig, ax = plt.subplots(figsize=(8, 5)) sns.barplot(data=df, x="name", y="gpa", hue="major", palette="Set2", ax=ax) ax.set_title("GPA by student & major", fontsize=14) ax.set_xlabel("Student") ax.set_ylabel("GPA") ax.set_ylim(0, 4.0) ax.spines[["top","right"]].set_visible(False) plt.tight_layout() plt.show()
🎨
Try this: Change palette="Set2" to palette="pastel" or palette="dark". Re-run. Palette choice is one of the most consequential design decisions in data visualization.
📊
Bar chart
Compare quantities across categories. Height = amount.
📈
Line chart
Show change over time. The slope tells the story.
⭕
Scatter plot
Show relationships between two variables. Each dot = one student.
🔥
Heatmap
Show intensity across two categories. Color = value.
04

Real-world data · ~8 min

Working with real data

Real grade distribution data from UIUC's public records. Thousands of sections across every department. This is what data scientists actually work with.

⏱ ~8 minutes · explore, don't just run

UIUC publishes historical grade distribution data publicly. We'll pull it directly from the internet using pandas. Each row is one course section with how many students received each letter grade.

Cell 6 — Load & explore real UIUC grade data
load_data.py
# Pull real UIUC grade distribution data (publicly available) url = "https://raw.githubusercontent.com/wadefagen/datasets/master/gpa/uiuc-gpa-dataset.csv" grades_df = pd.read_csv(url) print(grades_df.shape) # (rows, columns) print(grades_df.head()) print(grades_df["Subject"].value_counts().head(10)) # top 10 depts print(grades_df.describe())
💡
What you'll see: Each row is one course section from one semester. Columns include Subject, Number, Year, Term, and the count of students who received each grade (A+, A, A-, B+, etc.).
Cell 7 — Scatter: class size vs A-rate across departments
scatter.py
# Focus on design-adjacent departments depts = ["ART", "FAA", "ECE", "MATH", "ARCH"] dept_colors = { "ART": "#E8593C", "FAA": "#3B8BD4", "ECE": "#1D9E75", "MATH": "#BA7517", "ARCH": "#7F77DD" } # Calculate total students and percentage who got an A g = grades_df[grades_df["Subject"].isin(depts)].copy() g["total"] = g[["A+","A","A-","B+","B","B-","C+","C","C-","D+","D","D-","F"]].sum(axis=1) g["a_rate"] = (g["A+"] + g["A"] + g["A-"]) / g["total"] * 100 g = g[g["total"] >= 10].dropna() fig, ax = plt.subplots(figsize=(9, 6)) sns.scatterplot( data=g, x="total", y="a_rate", hue="Subject", style="Subject", palette=dept_colors, alpha=0.7, s=60, ax=ax ) ax.set_title("UIUC · Class Size vs A-grade Rate by Dept") ax.set_xlabel("Students in Section") ax.set_ylabel("% Receiving A (A+/A/A-)") ax.spines[["top","right"]].set_visible(False) plt.tight_layout() plt.show()
✨
Notice: Do bigger classes have lower A-rates? Do design departments cluster differently from ECE or MATH? Try changing the list in depts to include departments you're curious about, for example "PHYS", "ENGL", or "MUS".
05

The capstone · ~12 min

The finale

A four-panel publication-quality figure exploring UIUC grade distributions. The kind you see in research papers and data journalism. Yours to download.

⏱ ~12 minutes · read the code before running it
★  Capstone piece!! Save & share this one

Read through the entire code first. Every section is commented. By the time you run it, you should understand every line.

Cell 8 — The final publication-quality figure
final_visualization.py
# ── Setup ──────────────────────────────────────────────────────── DEPT_COLORS = { "ART": "#E8593C", # rust — art dept "FAA": "#3B8BD4", # blue — fine & applied arts "ECE": "#1D9E75", # teal — computer engineering "MATH": "#BA7517", # amber — math "ARCH": "#7F77DD" # purple — architecture } DEPTS = list(DEPT_COLORS.keys()) # Reload and prep the full UIUC dataset url = "https://raw.githubusercontent.com/wadefagen/datasets/master/gpa/uiuc-gpa-dataset.csv" df_all = pd.read_csv(url) df_all["total"] = df_all[["A+","A","A-","B+","B","B-","C+","C","C-","D+","D","D-","F"]].sum(axis=1) df_all["a_rate"] = (df_all["A+"] + df_all["A"] + df_all["A-"]) / df_all["total"] * 100 df_d = df_all[df_all["Subject"].isin(DEPTS)].dropna(subset=["a_rate","total"]) df_d = df_d[df_d["total"] >= 10] sns.set_theme(style="white", font_scale=0.9) # ── 2×2 grid layout ────────────────────────────────────────────── fig = plt.figure(figsize=(14, 10)) gs = fig.add_gridspec(2, 2, hspace=0.45, wspace=0.35) ax1 = fig.add_subplot(gs[0, 0]) ax2 = fig.add_subplot(gs[0, 1]) ax3 = fig.add_subplot(gs[1, 0]) ax4 = fig.add_subplot(gs[1, 1]) # A · Scatter: class size vs A-rate sns.scatterplot(data=df_d, x="total", y="a_rate", hue="Subject", style="Subject", palette=DEPT_COLORS, alpha=0.55, s=40, ax=ax1, legend=False) ax1.set_title("A · Class size vs A-grade rate", fontweight="bold") ax1.set_xlabel("Students in section"); ax1.set_ylabel("% receiving A") ax1.spines[["top","right"]].set_visible(False) # B · Violin: A-rate distribution per dept sns.violinplot(data=df_d, x="Subject", y="a_rate", palette=DEPT_COLORS, inner="quartile", ax=ax2) ax2.set_title("B · A-rate distribution by department", fontweight="bold") ax2.set_xlabel("Department"); ax2.set_ylabel("% receiving A") ax2.spines[["top","right"]].set_visible(False) # C · Heatmap: avg A-rate by year × dept pivot = df_d.pivot_table(values="a_rate", index="Year", columns="Subject", aggfunc="mean") sns.heatmap(pivot, annot=True, fmt=".1f", cmap="YlOrRd", linewidths=0.5, ax=ax3) ax3.set_title("C · Avg A-rate by year & department", fontweight="bold") ax3.set_xlabel("Department"); ax3.set_ylabel("Year") # D · Histogram: A-rate frequency per dept for dept, col in DEPT_COLORS.items(): sub = df_d[df_d["Subject"] == dept] ax4.hist(sub["a_rate"], bins=20, alpha=0.5, color=col, label=dept, edgecolor="white", linewidth=0.5) ax4.set_title("D · How often each dept gives A-grades", fontweight="bold") ax4.set_xlabel("% of class receiving A"); ax4.set_ylabel("Number of sections") ax4.legend(title="Dept") ax4.spines[["top","right"]].set_visible(False) # ── Titles & save ───────────────────────────────────────────────── fig.suptitle("UIUC Grade Distribution Analysis · ART / FAA / ECE / MATH / ARCH", fontsize=16, fontweight="bold", y=1.02) fig.text(0.5, 0.985, "Data: UIUC GPA Dataset (Wade Fagen-Ulmschneider) · Visualization by [your name]", ha="center", fontsize=9, color="#999") plt.savefig("uiuc_grade_analysis.png", dpi=200, bbox_inches="tight", facecolor="white") plt.show() print("✔️ Saved! Files panel → right-click → Download")
💾
To download: Click the folder icon in Colab's left sidebar → find uiuc_grade_analysis.png → right-click → Download. That's a 200 DPI image ready for print or portfolio.
